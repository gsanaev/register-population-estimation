# Register-Based Population Estimation in R

This project demonstrates a reproducible workflow for integrating synthetic administrative registers and estimating population counts using activity-based signals.

The workflow reflects methodological challenges in register-based population statistics, including:
- data integration across administrative sources
- validation and consistency checks
- identification of inactive records ("Lebenszeichen" concept)
- estimation of population overcoverage

Structure:
- data/: synthetic datasets
- R/: modular processing pipeline
- output/: aggregated results and visualizations
